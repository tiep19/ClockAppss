# ⏰ ClockApp - Ứng dụng Đồng hồ Đa năng

Đây là một ứng dụng Android được phát triển bằng **Kotlin**, tập trung vào các tính năng quản lý thời gian cơ bản và giao diện người dùng thân thiện.

##  Tính năng nổi bật
* Báo thức (Alarm):Thiết lập báo thức linh hoạt, có thông báo.
* Đồng hồ Thế giới (World Clock):Xem giờ tại nhiều thành phố lớn.
* Bấm giờ (Stopwatch):Đo thời gian chính xác với tính năng lưu vòng (Laps).
* Hẹn giờ (Timer):Đếm ngược thời gian đơn giản.


##  Công nghệ sử dụng
* **Ngôn ngữ:** Kotlin
* **Giao diện:** XML, ViewBinding
* **Thư viện:** Google Maps API (cho phần chọn thành phố), Material Design
